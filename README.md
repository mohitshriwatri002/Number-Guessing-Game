# Number Guessing Game

A console-based number guessing game built with Core Java.

## About
The program generates a random number between 1-100.
The user has to guess the correct number based on Higher/Lower hints.

## Technologies Used
- Java (Core Java)
- OOP Concepts
- Random Class, Scanner Class

## Manual Testing
25 test cases were designed and executed covering:
- Positive scenarios
- Boundary Value Analysis (BVA)
- Negative scenarios
- Edge cases
- UI/UX and Performance testing

**10 bugs were identified and documented.**

Testing report: guessing number report.xlsx

## How to Run
1. Clone this repository
2. Open NumberGuessingGame.java in any Java IDE
3. Run the main method
